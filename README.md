# Cleaner Robot

## How to install

Make sure you have node and latest version of npm installed

`node -v # v 10.0.0`

`npm -v # v 6.0.0`

Clone project

`git clone git@github.com:caki0915/robot-cleaner-typescript.git && cd robot-cleaner-typescript`

Install dependencies

`npm install`

To run the Cleaner

`npm start`

To run the test suite (Jest)

`npm test`
