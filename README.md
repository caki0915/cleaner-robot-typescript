# Cleaning Robot

## How to install

Make sure you have node and latest version of npm installed
`node -v # v 10.0.0`
`npm -v # v 6.0.0`

Clone project
`git clone git@bitbucket.org:carkih/cleaning-robot.git && cd cleaning-robot`

Install dependencies
`npm install`

To run the Cleaner
`npm start`

To run the test suite (Jest)
`npm test`
